# Gerenciador de Tarefas em Python

Aplicação simples em Python para gerenciamento de tarefas via terminal, com foco em organização de código, orientação a objetos e manipulação de dados.

## Funcionalidades
- Adicionar tarefas
- Listar tarefas (pendentes ou concluídas)
- Marcar tarefas como concluídas
- Remover tarefas
- Persistência de dados em arquivo JSON
- Ordenação por data de vencimento

## Conceitos Aplicados
- Programação orientada a objetos (classes e métodos)
- Manipulação de datas com datetime
- Leitura e escrita de arquivos JSON
- Validação básica de entradas
- Separação de responsabilidades

## Tecnologias Utilizadas
- Python 3
- Bibliotecas padrão: datetime, json

## Execução do Projeto
```bash
python main.py
